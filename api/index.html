How to use the API?

Make an GET Request to the page https://atjontv.github.io/AAAB/api/api.php

The Following requests are possible:

  • (1) https://atjontv.github.io/AAAB/api/api.php?uuid=<minecraft uuid>

Return values of the Requests

  • (1) Returns Boolean 'success', 'blacklisted' Integer 'uuid' String 'name', 'uuid'
